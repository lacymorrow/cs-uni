Zap Fitness: Front–end

Description

The Zap Fitness "Fitness Schedule" website provides a common area for Zap Fitness members and trainers to interact outside of the gym. Trainers may assign fitness schedules to users of different workout types. Users will see these fitness schedules output in a nice web calendar, and have access to update their actual workout so that they may compare it to their trainers recommendations. User profiles will contain basic health information about each person, updated as time progresses, and gives users the option of viewing and entering their "personal bests". Stretch features may include maps with overlays, garmin integration, workout analytics and graphs, and a posting or "popular public workouts".

Features

  • Trainer/User login
  • Trainer Admin - may assign user workouts/schedules
  • User Home - see trainer notes, user notes, updates
  • User Calendar - view trainer recommendations, enter actual workouts
  • User Admin/profile - contains user health stats/analytics/personal bests

The users of this site will be the members and trainers of Zap Fitness
